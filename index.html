AI-Driven COBOL Reverse Engineering Platform

Architected and implemented an AI-Powered platform for reverse engineering legacy COBOL systems into structured, analyzable representations, bridging compiler theory with modern agent-based AI Architectures. This project is developed independently, outside of my employment and client responsibilities.

  • Built compiler-style analysis pipelines (AST, ASG, CFG, DFG, PDG) to extract deterministic program structure and semantics;
  • Designed graph-based representations for cross-program dependency analysis and system-level understanding;
  • Implemented agent-driven workflows using LangGraph to orchestrate multi-step analysis and reasoning pipelines;
  • Integrated LangChain for controlled LLM interactions, enabling selective AI usage while preserving cost efficiency and determinism;
  • Designed and implemented an MCP-based modular tooling architecture, enabling scalable tool execution, and future extensibility;
  • Added enterprise-grade observability and execution metadata (timing, runs, errors) to support debugging and operational insights.

The platform combines Compiler Theory, Graph-Based Program Analysis, and selectively orchestrated LLM Workflows, and is exposed via Model Context Protocol (MCP) servers consumable by MCP-compatible clients such as Cursor and Claude.
